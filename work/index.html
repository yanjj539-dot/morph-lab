METHOD / SHARED RULES

DIFFERENT OUTPUTS.
ONE STANDARD.

无论最终形态是档案、图鉴还是操作台，所有系统都要保持内容可读、交互可解释、性能可降级、结果可部署。

  1. 01Content before effects

    先确定用户需要理解什么，再设计视觉事件。

  2. 02Motion with a reason

    每个运动都对应状态、层级或叙事变化。

  3. 03Fallback is a feature

    低性能设备得到更轻的体验，而不是残缺体验。

  4. 04Prototype in reality

    用可运行版本验证判断，不用静态想象替代反馈。
